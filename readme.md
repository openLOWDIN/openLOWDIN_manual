Installation notes.
=============

### Prerequisites: ###

* Sphinx 
* pdflatex

### Compile html ###

* run `make html`

### Compile pdf(LaTeX) ###

* run `make latexpdf`

### clean ###

* run `make clean`


### Tips ###
Check whitespace and empty lines 
Links to Sections in the Same Document requires full title header declared in the file
